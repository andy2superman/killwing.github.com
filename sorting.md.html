可视化需要harmony支持，在Chrome中开启：[chrome://flags/#enable-javascript-harmony](chrome://flags/#enable-javascript-harmony) ## 选择排序 <svg id="selection"></svg> ``` function selection(a) { for (let i = 0; i != a.length-1; ++i) { let t = i; // min pos // select minimum to front for (let j = i+1; j != a.length; ++j) { if (a[j] < a[t]) { t = j; } } swap(i, t); } } ``` ## 插入排序 <svg id="insertion"></svg> ``` function insertion(a) { for (let i = 1; i != a.length; ++i) { let t = a[i]; // insert current a[i] to front proper pos let j = i; for (; j > 0; --j) { if (a[j-1] > t) { a[j] = a[j-1]; } else { break; } } a[j] = t; } } ```
