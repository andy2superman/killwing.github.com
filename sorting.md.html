可视化需要harmony支持：Chrome中[开启harmony](chrome://flags/#enable-javascript-harmony) ## 选择排序 <svg id="selection"></svg> ``` function selection(a) { for (let i = 0; i != a.length-1; ++i) { var t = i; // min pos // select minimum to front for (let j = i+1; j != a.length; ++j) { if (a[j] < a[t]) { t = j; } } swap(i, t); } }; ```
